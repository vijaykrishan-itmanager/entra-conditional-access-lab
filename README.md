# Microsoft Entra Conditional Access Lab

## Objective
This lab simulates enterprise-level Conditional Access and Zero Trust configuration
inside a Microsoft 365 Developer environment.

## Lab Scope

- Role-Based Access Control (RBAC)
- Privileged Identity Protection
- Multi-Factor Authentication Enforcement
- Conditional Access Policy Testing
- Legacy Authentication Blocking
- Location-Based Access Restrictions
- Device Compliance Enforcement

## Zero Trust Model Applied

✔ Verify explicitly  
✔ Use least privilege access  
✔ Assume breach  

## Test Scenarios

1. Admin login from non-compliant device
2. MFA enforcement for privileged roles
3. Block legacy authentication protocols
4. Simulate risky sign-in
5. Monitor audit logs for policy triggers

## Tools Used

- Microsoft Entra ID
- Microsoft 365 Security Center
- Conditional Access Policies
- Audit Logs Monitoring
